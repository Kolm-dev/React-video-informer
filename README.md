
# Video informer
## Movie title info finder




## Installation




Install dependencies
```bash
yarn
```
Development mode
```bash
yarn dev 
```
Production mode
```bash
yarn build
```

## Technology

**Client:** React.JS






## Features

Clicking on the ID will show more information on a particular result
